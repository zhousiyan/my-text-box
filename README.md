# my-text-box

## 原生js文本下拉菜单显示  
## 使用步骤  
1. 编写HTML  
~~~html
<form class="box">
        <textarea id="editor" class="editor"></textarea>
        <div class="track"></div>  
~~~
2. 引入css  
~~~html
    <link rel="stylesheet" href="./index.css">
~~~
3. 引入js
~~~html
<script src="./index.js"></script>
~~~
